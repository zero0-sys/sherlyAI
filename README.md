# SherlyAI

Sherly AI adalah kecerdasan buatan berbentuk pacar/teman (*companion*) yang memiliki persona sangat emosional, bisa mengambek, inisiatif melempar topik, dan mengingat seluruh konteks obrolan masa lalu (persisten).

## Fitur Utama:
- **UI Hitam-Putih Minimalis**: Memaksimalkan fokus pembacaan obrolan tanpa distraksi visual.
- **Suara Sinhronous (Google TTS)**: Selalu berbicara dengan animasi gelembung '*real time*'.
- **Consciousness Engine**: Jika pengguna mengabaikannya, AI tidak akan sekadar menunggu; ia akan inisiatif *spam* chat, curhat, maupun mengirim '*Secret Code*' `[OFFLINE]` bila marah/acuh.
- **Persistent Local Memory**: Menyimpan memori jangka panjang melalui *localStorage*.

## Menjalankan Proyek
```bash
npm install
npm run dev
```

> Jangan lupa melampirkan `.env` beserta `VITE_GROQ_API_KEY` untuk mengaktifkan otak Llama 3.1.
